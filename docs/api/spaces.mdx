## Upload a Profile Picture
**<Text color="#F6AD55">POST</Text>** $API/profile

**Picture Structure**
<StructTable>
    <tr>
        <td>images</td>
        <td>File</td>
        <td>Image to upload (limited to 1 file)</td>
    </tr>
    <tr>
        <td>hash?</td>
        <td>SHA256</td>
        <td>
            Hash verification string. Not providing this will mark the change as "Unverified", and will leave
            "serverSideEncryption" as null
        </td>
    </tr>
</StructTable>
